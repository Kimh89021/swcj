# SW Auto Attributes

## 功能说明
- 从零件/装配体文件名中按字符位置截取字符串（不含扩展名），写入到自定义属性。
- 仅处理零件与装配体，图纸会通过链接的属性自动获取。
- 字符位置为 1 起始，结束位包含在内。
- 超过文件名长度的结束位会自动截断。
- 支持同时配置多个属性规则（固定值或文件名区间）。
- 打开文件时自动应用规则，确保属性最新。

## 使用
1. 打开零件或装配体。
2. 在菜单或工具栏找到“文件名属性 -> 设置/执行”。
3. 添加多条规则：
   - 类型=固定值：例如 属性名=作者，固定值=张三
   - 类型=文件名区间：例如 属性名=编号，起始位=1，结束位=12
   - 结束位=0 表示到末尾
4. 点击“保存并应用”，会立即写入当前模型的自定义属性。
5. 之后打开文件会自动应用规则，保持属性最新。
